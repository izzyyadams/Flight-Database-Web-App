Overview:
This project is a Python-based airline simulation that allows users (depending on their role) to explore flight operations, scheduling, and booking. The goal of this project was to build a functional, interactive simulation that models aspects of real-world airline management that interacts with a database.

Functionality:
Employee User:
-View and create flights
-Change status of flights
-Add new airplanes and airports
-Schedule plane maintenance
-View flight ratings, frequent customers, and earned revenue
-Login/Logout

Customer user:
-View customer's flights
-Search for and purchase flights
-Cancel trips
-Leave ratings and comments for flights
-Track spending
-Login/Logout

Tech Stack:
Python, using Flask
HTML, CSS, JavaScript
MySQL
